# CadasPatas

### Resumo:

O projeto consiste no desenvolvimento de uma aplicação voltada para ONGs de resgate de animais, com o objetivo de facilitar a gestão e a divulgação de pets disponíveis para adoção. A plataforma permitirá que as organizações registrem animais resgatados, incluindo informações como nome, idade, pelagem, estado de saúde e histórico. Além disso, a aplicação possibilitará disponibilizar esses animais para adoção, tornando os dados acessíveis para integração com sites, aplicativos ou sistemas parceiros. A solução busca organizar as informações, otimizar processos internos das organizações e ampliar a visibilidade dos animais, aumentando as chances de adoção responsável.
